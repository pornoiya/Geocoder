С использованием баз данных.

Автор: Кихтенко Татьяна (leninanadejda@gmail.com)

Описание
Данное приложение позволяет найти медиану координат, объекта по адресу. В качестве базы данных используется
вся Россия, скачано через ОSM.
Логика приложения реализована в модуле 'geocoder.py'. На приложениt написаны тесты, их можно найти в файле
'test_geocoder.py'.
Сначала происходит парсинг файла на две базы данных: база "адрес, ссылка" и база "ссылка, координаты". Происходит
подсчет медианного значения, и возвращается результат.

!!!Процесс парсинга занимает до 20 минут для одной области!!!

Требования
Python версии не ниже 3.6


Взаимодействие
Запуск через консоль. -a -- адрес в свободной форме, -p путь к файлу, где хранится файл osm всей России.
пример:
    geocoder.py -a "калининград, улица канта, 1" -p "path_to_osm_file"
